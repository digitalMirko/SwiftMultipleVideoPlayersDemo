# SwiftMultipleVideoPlayersDemo
Swift - Swift Multiple Video Players Demo - Demo 30 of 30

App created in Xcode 11.2.1 using Swift 5.1, iOS Deployment Target 13.2.2 and tested on a iPhone 11 Pro Max Emulator

In this demo app we demonstrate how to code a Swift Video Player Demo App.

There are so many interesting things within Swift. This demo shows one way to code a Swift Video Player device. There are
three different types of options demonstated. The first is the ability to play a local video that is saved with the application.
The second is to stream a video from its website. Then the third is to play a YouTube video on your device.

The buttons for the local and streaming options are marked. Then the box which was red in the storyboard but when running
on an app shows a clip of the YouTube clip is marked with a label to click the box to play the video.

The sample video saved and streamed (from the site) were found on file-examples.com. The YouTube clip was a demo sample
file with assorted amazing clips shared from YouTube directly.
